The free Body Mass Index calculator, also known as BMI, computes and classifies BMI for children and adults using data obtained from WHO and CDC.

[https://www.calculator.io/bmi-calculator/](https://www.calculator.io/bmi-calculator/)

== Usage ==

Add the BMI Calculator shortcode to your page, post or sidebar:

`[calcio_bmi_calculator]`

Add the following code to your template where you would like the BMI Calculator to appear:

`<?php display_calcio_bmi_calculator(); ?>`

== Installation ==

1. Upload the BMI Calculator /calcio_bmi_calculator/ folder to the /wp-content/plugins/ directory.
2. Activate the BMI Calculator plugin through the "Plugins" menu in WordPress.
